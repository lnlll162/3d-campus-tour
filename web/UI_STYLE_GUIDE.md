# TJCU 风格 UI 规范（本项目）

> 目标：将本项目所有 UI（控制面板、信息面板、按钮、开关、弹层、Toast、加载、首页等）的**颜色与视觉气质**统一到天津财经大学官网（https://www.tjcu.edu.cn/）的“高校门户”风格：稳重的深蓝主色、干净的浅色背景、克制的阴影与圆角、清晰的信息层级。

## 1. 颜色体系（Color System）

### 1.1 核心原则

- **主色（Primary）**：学校官网风格的深蓝，承载所有主要交互（主按钮、强调、可点击高亮）。
- **辅助色（Secondary）**：主色的加深版本，用于 hover/active 与强调边框。
- **导航色（Nav）**：更深的蓝（接近官网顶部导航条），用于顶部栏、重要区域标题。
- **中性色（Neutrals）**：浅灰背景、灰蓝文本，用于卡片底色、分割线、次要文本。
- **语义色（Semantic）**：错误/成功/警告（保持克制，不使用高饱和荧光色）。

### 1.2 变量命名（项目统一使用 CSS Variables）

建议在 `web/src/styles/main.css` 的 `:root` 里统一维护：

- `--primary-color`：主色
- `--primary-hover`：主色 hover
- `--secondary-color`：主色 active/更深
- `--nav-bg`：导航深蓝
- `--bg-color`：页面背景
- `--text-color`：主文本
- `--muted-text`：次级文本
- `--card-bg`：卡片底色（浅）
- `--card-border`：卡片边框
- `--divider`：分割线
- `--shadow`：阴影

> 备注：当前项目已经有部分变量（`--primary-color` / `--nav-bg` 等），后续只做补齐与统一引用。

### 1.3 推荐色值（可微调，以官网视觉为准）

> 由于本环境无法直接抓取官网 CSS，我先提供一套“高校官网深蓝系”的推荐值，后续你如果希望我做到 1:1，可以你截图/提供官网 CSS 片段，我再精确对齐。

- 主色 `--primary-color`: `#0B2F6B`
- Hover `--primary-hover`: `#083062`
- Active/Secondary `--secondary-color`: `#062550`
- 导航 `--nav-bg`: `#072A5A`
- 背景 `--bg-color`: `#F5F7FA`
- 主文本 `--text-color`: `#0F172A`
- 次文本 `--muted-text`: `#64748B`
- 卡片底 `--card-bg`: `rgba(255,255,255,0.92)`
- 卡片边 `--card-border`: `rgba(15,23,42,0.08)`
- 分割线 `--divider`: `rgba(15,23,42,0.08)`
- 阴影 `--shadow`: `0 12px 30px rgba(15, 23, 42, 0.10)`

## 2. 字体与排版（Typography）

### 2.1 字体

- 默认：系统 UI 字体栈（已存在）
- 数字/性能信息：等宽字体（已存在）

### 2.2 字号层级

- 页面/模块标题（如“控制面板”）：14px~16px，font-weight 600~700
- 卡片标题（如“视角模式”“相机控制”）：12px，font-weight 600~650
- 表单/按钮：13px
- 次要说明/Hint：11px~12px

### 2.3 文案风格

- 简短、明确、偏“门户系统”风格
- 避免花哨 emoji（项目内目前少量使用在 toast/欢迎弹窗里，后续应统一移除或保留极少）

## 3. 组件风格（Components）

### 3.1 卡片（control-group / panel-info 等）

- 背景：浅色/半透明白
- 边框：细线，低对比
- 圆角：12~16
- 阴影：克制（主要由外层面板承担，内部卡片可仅边框即可）

### 3.2 按钮

- 主按钮：深蓝底 + 白字
- Hover：略加深，轻微抬起（translateY(-1px)）
- Active：回落
- 圆角：10
- 禁止使用高饱和渐变；如使用渐变，也必须非常克制（接近纯色）

### 3.3 Select / Input

- 白底或浅白半透明
- focus ring：使用主色的低透明度外发光（3px）
- 高度统一：36px（控制面板内）

### 3.4 Checkbox

- accent-color 使用主色
- 默认两列；“显示选项”可按需求单列

### 3.5 Toast / 弹层

- Toast：顶部居中，圆角 8~10，阴影克制
- 弹层：遮罩 + 中心卡片（目前欢迎弹窗已移除），色彩克制

## 4. 布局与响应式（Layout & Responsive）

### 4.1 桌面端

- 顶部居中固定面板：宽度 `min(1200px, 100vw - 24px)`
- 展开后：内容区为栅格（4列）+ 信息区（180px）

### 4.2 窄屏/移动端

- 面板仍在顶部，默认折叠
- 展开后：内容区横向滑动（scroll-snap），信息区下移
- 滑动提示：左右渐隐遮罩（已实现）

## 5. 实施清单（你可以用来验收）

- 控制面板：颜色、边框、阴影与官网一致
- HomeButton/返回首页按钮：颜色一致
- InfoPanel：标题与按钮用主色体系
- LoadingScreen：背景/进度条使用主色体系
- Toast：成功/错误颜色克制，不刺眼

## 6. 变更流程建议

- 先只改 `:root` 的主题变量，确保全局主色统一
- 再逐个组件把 hard-coded 色值替换成变量引用
- 最后统一做对比度检查（WCAG 基本可读性）
